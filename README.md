# 6-practica
11)Баланс нөлге жеткенде: шоттағы баланс 0-ге жеткен жағдайда автоматты түрде хабарлап, сол шотты жабу.

12)Тізім немесе сөздік арқылы сақтау: студенттер мен шоттарды объект ретінде тізім немесе сөздікке сақтап, қайталануларды тексеру.

26)Бірнеше студенттің орташа жасын есептеу: базадағы барлық студенттің жасын жинап орташа есептеу.

6)Профиль ақпаратын жаңарту: пайдаланушы аты, телефон немесе суретін өзгерткенде property әдістерін қолданып, қателерді өңдеу.
